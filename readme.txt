Easily add Google Ads conversion tracking codes on your website.

== Description ==

This plugin will let you add Google Ads conversion codes given by Google.
This includes adding scripts in the head section of your website, to display them on specific pages, checkout or contact form (contact form 7)


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/conversion_helper` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings->Conversion Helper 
4. Fill in all details as given by Google Ads
5. Go to relevant pages edit and find "Conversion helper" box, where you can enter relevant conversion codes to fire on specific pages.
